# Chaînes de production

Comme déjà birevement expliqué dans “[Comment Bien débuter Sur Stingray](Bien%20de%CC%81buter%20sur%20Stingray%204fd8e62a17fe4172ac96ae45e4e4babf.md)”, Pour faire de l’argent et obtenir des produits en tout genres il sera nécéssaire de passer par les chaînes de production. 

Une chaîne de production est l'ensemble des opérations de fabrication et de transformation nécessaires à la réalisation d'un produit manufacturé en partant de matières premières jusqu'à la mise en vente sur le marché.

Sur le serveur vous retrouverez 3 étapes clés des chaines de production :

- la production ou récupération de matière première
- la(les) transformation(s) nécéssaire(s) pour créer un produit manufacturé (produit fini)
- L’export ou la vente de produit manufacturé

La production de matière première se réalise auprès des machines de production, ce sont des machines qui vont généré une quantité infinie de produit initial. Par exemple, la pompe à eau  produit de l’eau sale en quantité illimité.

La récupération de matière première se réalise “à la main”. Cela regroupe par exemple, la récolte, l’abattage, le minage, la chasse, la pêche.

La Transformation se réalise auprès des machines de transformations. Ces machines ont pour but de prendre un ou plusieurs produit, ces produits seront consommé pour réaliser un produit manufacturé. Il est possible, dans certaines chaînes de production, d’avoir plusieurs transformations consécutive sur des machines différentes.

L’export se réalise auprès des NPCs prévu a cet effet. Plus d’informations sur la page [Achats et Ventes d’items](Achats%20et%20Ventes%20d%E2%80%99items%20e9ce708caa904ffa95e8a2508c77cce5.md).

Machine de Production :

La machine de production se compose de 2 parties:

- un Panel : partie permettant d’interagir avec la machine
- un Output (Sortie) : Espace de stockage qui récupère les produits créer par la machine

Machine de Transformation :

La machine de transformation se compose de 3 parties:

- Un Panel : partie permettant d’interagir avec la machine
- Un Input (Entrée) : Espace de stockage qui stocke les produits qui seront consommé lors de la transformation
- un Output (Sortie) : Espace de stockage qui récupère les produits transformé
